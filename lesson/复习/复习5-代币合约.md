## ERC20标准的组成

## 6个核心函数
* 查询函数：totalSupply、balanceOf、allowance; 查询类函数不会改变合约状态，所以被标记为view
* 操作函数：transfer、approve、transferFrom

2个必需事件

* Transfer事件：记录转账
* Approval事件：记录授权

3个可选元数据

* name：代币名称
* symbol：代币符号
* decimals：小数位数

```sol
// 查询总供应量
totalSupply: function totalSupply() public view returns (uint256)
// 查询余额
// account: 要查询的账户地址
balanceOf: function balanceOf(address account) public view returns (uint256)
// 查询授权额度-返回被授权人可以使用授权人的代币数量
allowance: function allowance(address owner, address spender) public view returns (uint256)

// 操作类函数会改变合约状态，不能标记为view或pure。
// 直接转账
transfer: function transfer(address to, uint256 amount) public returns (bool)

// 授权
approve: function approve(address spender, uint256 amount) public returns (bool)

// 授权转账
transferFrom: function transferFrom(address from, address to, uint256 amount) public returns (bool)
```
