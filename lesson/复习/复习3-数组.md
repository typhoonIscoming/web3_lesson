# 定长数组
```sol
uint[5] public numbers;
```
* 不支持push
* 不支持pop
* delete删除某个元素，这个元素变成0，length不变

## 动态数组
```sol
uint[] public numbers;
```
* 支持push
* 删除最后元素（pop）
* delete numbers--清空数组，length变成0

## 多维数组
```sol
// 定长二维数组
uint[3][4] public fixedMatrix;  // 4行，每行3个元素
// 混合数组
uint[][5] public mixedArray;  // 5个动态数组
uint[3][] public mixedArray2;  // 动态数量的定长数组

uint[3][4]
     ↑  ↑
     |  |
     |  └─ 4个
     └──── 长度为3的数组
```
