# 定长数组
```sol
uint[5] public numbers;
```
* 不支持push
* 不支持pop
* delete删除某个元素，这个元素变成0，length不变

## 动态数组
```sol
uint[] public numbers;
```
* 支持push
* 删除最后元素（pop）
* delete numbers--清空数组，length变成0

## 多维数组
```sol
// 定长二维数组
uint[3][4] public fixedMatrix;  // 4行，每行3个元素
// 混合数组
uint[][5] public mixedArray;  // 5个动态数组
uint[3][] public mixedArray2;  // 动态数量的定长数组

uint[3][4]
     ↑  ↑
     |  |
     |  └─ 4个
     └──── 长度为3的数组
```

## mapping
Mapping只能作为状态变量，不能在函数内创建。
* Mapping的存储结构依赖于哈希计算
* Memory和calldata不支持这种存储方式
* Mapping必须在区块链上永久存储

## mapping不能作为参数或返回值

## 虽然不能删除整个mapping，但可以删除单个键的值(不是真正"删除"，而是"重置")

## ERC20授权流程：
1. 步骤1：Alice授权Bob可以花费100个代币
* alice.approve(bob, 100)
  ↓
* allowance[alice][bob] = 100

2. 步骤2：Bob代Alice转账50个代币给Carol
* bob.transferFrom(alice, carol, 50)
  ↓
* 检查：allowance[alice][bob] >= 50 ✓
  ↓
* 扣除授权：allowance[alice][bob] = 50
    ↓
* 转账：balances[alice] -= 50, balances[carol] += 50

结果：
- Alice余额减少50
- Carol余额增加50
- Bob剩余授权额度50
